デモ

特定の日時までの残り時間をカウントダウン

2020年 東京オリンピックまで...  0日 00時間 00分 00.0秒

期日を過ぎているときはメッセージを表示

1964年 東京オリンピックまで...  0日 00時間 00分 00.0秒

ダウンロード

  • countdown.js
  • countdown.min.js

使い方

ステップ1: プラグインを読み込み

<script src="js/jquery.js"></script>
<script src="js/countdown.js"></script>

ステップ2: HTML を記述

<p class="remain">
  <span class="day">0</span>日
  <span class="hour">00</span>時間
  <span class="minute">00</span>分
  <span class="second">00.0</span>秒
</p><!-- /.remain -->

ステップ3: 起動スクリプトを記述

<script>
jQuery(function($) {
  $('.remain').countdown({
    goal: '2020/7/24 08:00:00',
  });
});
</script>

オプション

全てのオプションとデフォルトの値

$('.remain').countdown({
  goal: 'yyyy/MM/dd HH:mm:ss',
                       // カウントダウンの期日 (※必須項目)

  digits: 1,           // 「残り秒」の小数点以下の桁数

  wrapper: {
    day:    '.day',    // 「残り日数」が表示されるタグのCSSセレクタ
    hour:   '.hour',   // 「残り時間」が表示されるタグのCSSセレクタ
    minute: '.minute', // 「残り分」が表示されるタグのCSSセレクタ
    second: '.second', // 「残り秒」が表示されるタグのCSSセレクタ
  },

  text: '期日を過ぎました。',
                       // 期日を過ぎたときに表示されるメッセージ
});
項目 必須？ デフォルト値 説明
goal ✔ - カウントダウンの期日。
yyyy/MM/dd HH:mm:ss の形式で入力すること。
digits   1 「残り秒」の小数点以下の桁数。
0 にすると小数点以下は表示されなくなる。
wrapper.day   .day 「残り日数」が表示されるタグのCSSセレクタ。
wrapper.hour   .hour 「残り時間」が表示されるタグのCSSセレクタ。
wrapper.minute   .minute 「残り分」が表示されるタグのCSSセレクタ。
wrapper.second   .second 「残り秒」が表示されるタグのCSSセレクタ。
text   「期日を過ぎました。」 カウントダウンの期日を過ぎたときに表示されるメッセージ。
